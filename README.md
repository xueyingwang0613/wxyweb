# 王雪莹个人网站

个人 AI 实践档案馆，采用原生 HTML、CSS 与 JavaScript 实现，无需安装依赖。

## 本地预览

```bash
python3 -m http.server 4173
```

然后访问 `http://127.0.0.1:4173`。

## 文件

- `index.html`：页面内容与结构
- `styles.css`：视觉系统、响应式布局与动效
- `script.js`：导航、筛选、案例展开与滚动交互
- `scripts/verify-site.cjs`：Playwright 桌面端 / 移动端渲染检查
- `PERSONAL_WEBSITE_PLAN.md`：网站规划文档

## 内容说明

网站内容基于王雪莹提供的个人描述与实践方向整理，不使用简历式写法，也不虚构项目结果。公开页面默认不展示手机号，只保留邮箱作为联系入口。
